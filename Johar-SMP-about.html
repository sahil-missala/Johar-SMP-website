About Johar SMP

Johar SMP is survival multiplayer minecraft server created and managed by Ibb201 [in game username] in the month of may when his 3 friends wanted to play minecraft together. Later on 3 more of his friends joined the smp and coined the name Johar SMP after 2 member’s first syllable from the names Joseph and Harshil.

Johar SMP Seasons

Johar SMP has gone through 3 seasons from which the first and second season are completed while the third one is still going on.

Season 1

Season 1 originally started when 4 friends were chatting on discord, when 1 of his friend suggested to make a Minecraft server to play on. So Ibb201 created the server on may 2nd 2025, beginning the start of Johar SMP. While originally not named Johar SMP rather called Ibb201’s Minecraft server but later renamed to Johar SMP.

Season 2

Season 2 begun on 25th June 2025, unlike the previous server this server added some new additions like a level up system giving players buffs and abilities. This server was also the most chaotic server as someone was caught using cheats, later when new members were introduced, these new members caused chaos leading to them removed. But a new member joined, who wasn’t chaotic. Due to the server being so chaotic and players progressing too fast, this caused the server die out fast.

Season 3

Season 3 begun on 15th August 2025, and was the most modded server. This server added guns, new enchantments and many more game changing mods that is being added as the server progresses. This server is also lasting the longest due to locking the end to prevent players from progressing too fast. This server had no new members added.
